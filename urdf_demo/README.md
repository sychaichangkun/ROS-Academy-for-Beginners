# urdf_demo

urdf机器人模型功能包。

## 运行方法


```sh
$ roslaunch urdf_demo display_urdf_link_joint.launch     #利用urdf制作机器人的link和joint,并在rviz中显示
$ roslaunch urdf_demo display_urdf_link_position.launch   #利用urdf制作机器人的link和position,并在rviz中显示
$ roslaunch urdf_demo display_urdf_color_geometry.launch     #利用urdf制作机器人，并在rviz中显示机器人
$ roslaunch urdf_demo display_xacro.launch           #利用xacro制作机器人，并在rviz中显示机器人
$ roslaunch urdf_demo display_gazebo_rviz.launch     #在gazebo和rviz中同时显示机器人
```
